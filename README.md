## Kingtp (First Version)

## Description
Kingto is the **first version** of my personal AI chatbot, integrate  Gemini API (v2.0 or latest).
It has a proper database to store **chat history**, supports **prompt personalization**, and can perform **text summarization** and understand user text.

This is the **first version** of Kingtp. Future versions will be improved with more unique features and enhanced AI capabilities.

## Features
-->AI chatbot powered by Gemini API
--> Stores chat history in a database
--> Prompt personalization for each user
--> Text summarization and comprehension
--> Simple frontend interface (HTML, CSS, JS)
--> First version of KingTP, scalable for future improvements

## Technologies Used
--> Python (Flask / Django / FastAPI)
--> Gemini API (v2.0 or latest)
--> MY SQL Workbench / SQLite (Database)
--> Frontend: HTML, CSS, JavaScript
